Project Name: Titanic Data Analysis

Description:
This project uses the Titanic dataset to practice data analysis in Python. The program reads the Titanic CSV file, checks the dataset information, shows missing values, and creates charts to understand passenger survival patterns.

The project analyzes information such as passenger age, fare, gender, passenger class, and survival status. It also includes visualizations such as scatter plots, histograms, bar charts, boxplots, count plots, and a correlation heatmap.

Language Used:
Python

Libraries Used:
pandas
numpy
matplotlib
seaborn

Files:
Titanic.py
titanic.csv

How to Use:
Download both files and keep them in the same folder. Run Titanic.py in PyCharm, VS Code, or another Python editor.
